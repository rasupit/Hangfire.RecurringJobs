# Hangfire.Extension v0.1.4

Patch release for self-contained UI styling.

## Highlights

- Inlines the recurring jobs UI stylesheet into the embedded layout
- Removes the runtime dependency on serving `hangfire-extension.css` from host static web assets
- Keeps the embedded UI styling stable when consumed from local package feeds and GitHub release assets

## Installation

```powershell
dotnet add package Hangfire.Extension --version 0.1.4
```

## Consumer Impact

- No host changes are required
- Hosts should continue to call `MapRazorPages()`
- The recurring jobs page no longer relies on an external stylesheet request at runtime

## Notes

- This package still does not configure Hangfire storage
- The recurring jobs UI remains intended for internal/admin usage
