# Hangfire.Extension v0.1.8

Patch release for recurring jobs action density and toolbar consistency.

## Highlights

- Normalizes the search action buttons so `Apply` and `Reset` share the same size and alignment
- Replaces row action text buttons with compact icon-only controls
- Uses hover titles and accessible labels for trigger, enable or disable, and edit cron actions

## Installation

```powershell
dotnet add package Hangfire.Extension --version 0.1.8
```

## Consumer Impact

- No host integration changes are required
- Hosts should continue to call `MapHangfireRecurringJobsApi()` and `MapRazorPages()`
- The recurring jobs page keeps the self-contained inline styling introduced in previous releases

## Notes

- This package still does not configure Hangfire storage
- The recurring jobs UI remains intended for internal/admin usage
