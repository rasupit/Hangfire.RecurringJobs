# Hangfire.Extension v0.1.3

Patch release for stylesheet URL rendering.

## Highlights

- Fixes the embedded recurring jobs layout so the stylesheet is emitted as an absolute resolved URL
- Prevents packaged hosts from requesting `hangfire-extension.css` as a broken relative path
- Adds regression coverage for unresolved `~/...` stylesheet links

## Installation

```powershell
dotnet add package Hangfire.Extension --version 0.1.3
```

## Consumer Impact

- No host changes are required
- Hosts should continue to call `MapStaticAssets()` and `MapRazorPages()`
- The recurring jobs UI should now load its stylesheet correctly in packaged consumer apps

## Notes

- This package still does not configure Hangfire storage
- The recurring jobs UI remains intended for internal/admin usage
