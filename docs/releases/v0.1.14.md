# Hangfire.Extension v0.1.14

Patch release for host-theme-friendly recurring jobs chrome.

## What's Changed

- Moves the recurring jobs header and general surfaces back toward standard Bootstrap classes so host themes can style them naturally
- Keeps only a thin custom layer for square icon action buttons
- Replaces unicode action glyphs with inline Bootstrap Icons-style SVGs and a tiny local `.bi` CSS snippet

## Host Integration

Hosts may continue layering stylesheets through `options.Styles`:

```csharp
builder.Services.AddHangfireRecurringJobs(options =>
{
    options.RoutePrefix = "/JobsAdmin";
    options.RequireAuthorization = true;
    options.Styles.Add("/lib/bootstrap/dist/css/bootstrap.min.css");
    options.Styles.Add("/css/site.css");
});
```
