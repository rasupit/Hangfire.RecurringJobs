# Hangfire.Extension v0.1.13

Patch release for multi-stylesheet host overrides.

## What's Changed

- Changes `RecurringJobsOptions.Styles` from a single stylesheet override to an ordered collection
- Allows hosts to layer multiple stylesheets, such as Bootstrap plus a site theme
- Keeps the embedded fallback theme when no override styles are configured

## Host Integration

Example:

```csharp
builder.Services.AddHangfireRecurringJobs(options =>
{
    options.RoutePrefix = "/JobsAdmin";
    options.RequireAuthorization = true;
    options.Styles.Add("/lib/bootstrap/dist/css/bootstrap.min.css");
    options.Styles.Add("/css/site.css");
});
```
