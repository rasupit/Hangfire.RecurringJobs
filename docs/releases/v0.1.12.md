# Hangfire.Extension v0.1.12

Patch release for recurring jobs action button density.

## What's Changed

- Restores compact icon-only action controls for trigger, enable/disable, and edit cron
- Keeps hover text through button tooltips and accessible labels through `aria-label`
- Preserves the embedded fallback stylesheet delivery from the previous release

## Host Integration

Hosts should continue to call:

- `MapStaticAssets()`
- `MapHangfireRecurringJobsApi()`
- `MapRazorPages()`

If `Styles` is not configured, the library uses its embedded fallback theme automatically.
