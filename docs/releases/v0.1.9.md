# Hangfire.Extension v0.1.9

Patch release for recurring jobs page simplification.

## Highlights

- Removes the redundant summary cards above the search area
- Keeps page count and total count only in the scheduled jobs section header
- Normalizes the search toolbar so `Apply` and `Reset` share the same size

## Installation

```powershell
dotnet add package Hangfire.Extension --version 0.1.9
```

## Consumer Impact

- No host integration changes are required
- Hosts should continue to call `MapHangfireRecurringJobsApi()` and `MapRazorPages()`
- The recurring jobs page keeps the compact icon action layout introduced in the previous release

## Notes

- This package still does not configure Hangfire storage
- The recurring jobs UI remains intended for internal/admin usage
