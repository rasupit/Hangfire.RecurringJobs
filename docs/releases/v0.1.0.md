# Hangfire.Extension v0.1.0

First public release of `Hangfire.Extension`.

## Highlights

- Embedded recurring jobs admin UI for ASP.NET Core hosts
- Recurring jobs API under the configured base path
- Code-based recurring job definition registration
- `SkipConcurrentExecutionAttribute` for overlap protection
- Host integration tests and SQLite-backed Hangfire tests

## Installation

```powershell
dotnet add package Hangfire.Extension --version 0.1.0
```

## Usage

```csharp
using Hangfire;
using Hangfire.Extension;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddAuthorizationBuilder()
    .AddPolicy("RecurringJobAdmin", policy => policy.RequireRole("Admin"));

builder.Services.AddHangfire(configuration =>
{
    configuration.UseSimpleAssemblyNameTypeSerializer();
    configuration.UseRecommendedSerializerSettings();
});

builder.Services.AddHangfireRecurringJobs("/recurring-jobs");

var app = builder.Build();

app.UseAuthorization();

app.MapHangfireRecurringJobsApi();
app.MapRazorPages();

app.Run();
```

Canonical UI route:

- `/recurring-jobs`

Canonical edit route:

- `/recurring-jobs/{id}/edit`

API base route:

- `/recurring-jobs/api/jobs/recurring`

## Optional: Register Definitions

```csharp
builder.Services.AddRecurringJobDefinition<ReportJobs>(
    "nightly-report",
    jobs => jobs.RunNightlyReport(),
    "0 2 * * *",
    TimeZoneInfo.Utc,
    "default");
```

## Optional: Prevent Overlapping Executions

```csharp
using Hangfire.Extension;

public sealed class ReportJobs
{
    [SkipConcurrentExecution(timeoutInSeconds: 30)]
    public Task RunNightlyReport()
        => Task.CompletedTask;
}
```

## Notes

- This package does not configure Hangfire storage.
- The host must still call `MapRazorPages()`.
- The host should call `MapStaticAssets()` so the embedded UI can load package-scoped assets from `/hangfire-extension/...`.
- The recurring jobs UI is intended for internal/admin usage.
- Package static web assets are isolated under `/hangfire-extension/...`, which avoids collisions with host `wwwroot` assets such as `lib/bootstrap`, `lib/jquery`, or `favicon.ico`.
