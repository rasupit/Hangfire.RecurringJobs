# Hangfire.Extension v0.1.15

Patch release for neutral Bootstrap surface inheritance.

## What's Changed

- Removes `bg-body` from the recurring jobs navbar and card header
- Prevents stock Bootstrap surface classes from forcing those sections into dark background treatments
- Keeps the hybrid theme approach from the previous release

## Host Integration

Hosts may continue layering stylesheets through `options.Styles`:

```csharp
builder.Services.AddHangfireRecurringJobs(options =>
{
    options.RoutePrefix = "/JobsAdmin";
    options.RequireAuthorization = true;
    options.Styles.Add("/lib/bootstrap/dist/css/bootstrap.min.css");
    options.Styles.Add("/css/site.css");
});
```
